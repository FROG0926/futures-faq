什么是期货？期货和股票有什么区别？
期货是一种标准化的远期合约，由期货交易所统一制定，约定在未来某一特定时间和地点交割一定数量标的物。与股票的核心区别：

1. 双向交易 — 期货可以做多（买涨）也可以做空（卖跌），涨跌都有盈利机会；股票通常只能做多。
2. T+0 交易 — 期货当天开仓当天即可平仓，不限次数；A股是T+1，当日买入次日才能卖出。
3. 保证金杠杆 — 期货只需缴纳5%-15%的保证金即可参与全额合约交易，杠杆放大收益也放大风险；股票通常是全额交易。
4. 合约到期日 — 期货有明确的到期日，到期必须平仓或交割；股票可长期持有。
5. 每日无负债结算 — 期货每日收盘后按结算价清算盈亏，资金实时划转。
2
什么是保证金？保证金比例一般是多少？
保证金是投资者参与期货交易时按合约价值一定比例缴纳的资金，用于保证履约。它让投资者可以用较少资金撬动大额合约，实现杠杆交易。

保证金比例因品种而异：
• 商品期货：通常为合约价值的 5%-15%（如玉米约8%，原油约7%-10%，黄金约8%-12%）
• 股指期货：交易所标准为 8%，期货公司通常上浮至10%-12%
• 保证金=合约价值×保证金比例

保证金分为初始保证金（开仓所需）和维持保证金（持仓所需，通常为初始保证金的75%-80%）。当账户权益低于维持保证金水平时，将收到追加保证金通知。
3
什么是开仓、平仓、持仓？
这三个是期货交易最基础的操作术语：

开仓（建仓）：新买入或新卖出一份期货合约，建立头寸。
• 买入开仓 = 做多，预期价格上涨
• 卖出开仓 = 做空，预期价格下跌

平仓：对已持有的合约进行反向操作，了结头寸。
• 买入平仓 = 了结空头持仓（之前卖出的买回来）
• 卖出平仓 = 了结多头持仓（之前买入的卖出去）

持仓：未平仓的合约头寸，即当前持有的仓位。持仓量是市场活跃度的重要参考指标。
4
什么是T+0交易？有什么优势和风险？
T+0交易是指当日开仓的合约当日即可平仓，且日内交易次数无限制。这是期货与A股（T+1）的核心区别之一。

优势：
• 资金周转效率高，抓住日内波动机会
• 可以及时止盈止损，遇到不利行情立即离场
• 适合短线交易策略

风险：
• 容易导致频繁交易，增加手续费成本
• 可能诱发情绪化操作、追涨杀跌
• 新手容易陷入赚小钱亏大钱的陷阱

建议：新手应克制日内交易频率，设定明确的交易计划和止损纪律。
5
什么是做多和做空？做空怎么赚钱？
做多（买入开仓）：预期价格会上涨，先买入合约，待价格上涨后卖出平仓，赚取差价。

做空（卖出开仓）：预期价格会下跌，先卖出合约，待价格下跌后买入平仓归还，赚取差价。

做空赚钱的逻辑：假设螺纹钢期货当前4000元/吨，你认为会跌到3800元。先卖出开仓1手（10吨），价格跌到3800元时买入平仓。差价200元×10吨=盈利2000元（不含手续费）。

这就是期货双向交易的优势——涨跌都能盈利，关键在于判断方向是否正确。

期货合约代码怎么解读？如RB2605是什么意思？
期货合约代码由品种代码 + 到期年月组成：

以 RB2605 为例：
• RB = 品种代码，代表螺纹钢（Rebar）
• 26 = 年份，2026年
• 05 = 月份，5月到期

常见品种代码速查：
• 金属：CU铜、AL铝、AU黄金、AG白银、NI镍
• 黑色：RB螺纹钢、HC热卷、I铁矿石、J焦炭、JM焦煤
• 能源化工：SC原油、FU燃料油、MA甲醇、TA精对苯二甲酸(PTA)、PG液化气
• 农产品：M豆粕、Y豆油、P棕榈油、C玉米、SR白糖、CF棉花
• 金融期货：IF沪深300、IC中证500、IH上证50、IM中证1000、T 10年国债

什么是主力合约？为什么要交易主力合约？
主力合约是指同一品种所有月份合约中成交量最大、持仓量最大的那个合约。期货软件中通常默认显示的即为该品种的主力合约。

交易主力合约的原因：
• 流动性好：买卖盘口活跃，挂单容易成交，滑点小
• 价差合理：买卖价差窄，交易成本低
• 价格代表性：最能反映市场对该品种的真实定价

主力合约会随着交割月临近而切换（移仓换月），一般个人投资者在交割月前一个月左右需平仓旧合约、开仓新主力合约。

期货价格是如何形成的？现货与期货价格有什么关系？
期货价格的核心形成机制是持有成本理论：

期货价格 = 现货价格 + 持有成本（利息、仓储、保险等）- 持有收益

在到期日，期货价格与现货价格会趋于一致（收敛），否则将产生无风险套利机会。

升贴水概念：
• 升水（Contango）：期货价格 > 现货价格，通常因仓储成本等
• 贴水（Backwardation）：现货价格 > 期货价格，可能反映现货供应紧张

基差 = 现货价格 - 期货价格，是套期保值中需要重点关注的指标。

期货开户需要什么条件？需要准备哪些材料？
个人投资者开立期货账户的基本条件：

条件要求：
• 年满18周岁，具有完全民事行为能力
• 无不良征信记录（非期货市场禁入者）
• 通过投资者适当性风险测评（通常需达到C3及以上等级）
• 具备基本的期货知识和风险承受能力

准备材料：
• 本人有效身份证原件
• 本人名下银行借记卡（用于银期转账，支持工农中建交等大型银行）
• 手写签名照（线上开户时拍照上传）

特定品种额外要求：
• 股指期货/国债期货/原油期货：需账户可用资金≥50万元（连续5个交易日），通过知识测试（80分以上），有10笔以上实盘交易记录或模拟交易经历
• 商品期权：需通过期权知识测试，完成模拟交易经历

期货开户流程是怎样的？线上可以开吗？
目前绝大多数期货公司支持线上开户，流程如下：

线上开户流程（约20-30分钟）：
1. 下载期货公司官方APP或通过客户经理专属链接进入开户页面
2. 上传身份证正反面照片、签名照
3. 填写个人基本信息、联系方式
4. 完成投资者适当性风险测评问卷
5. 视频见证认证（与期货公司工作人员远程视频）
6. 阅读并签署《期货经纪合同》《风险说明书》等协议
7. 申请交易编码（由期货公司向交易所申请）
8. 办理银期转账关联（绑定银行卡）

重要提示：
• 不要直接在官网/APP自助开户——默认手续费率通常是交易所的2-5倍，且后期难以调整
• 建议先联系客户经理协商手续费和保证金优惠后，走专属开户通道
• 开户免费，不需要任何费用
• 整个开户过程约1-2个工作日完成审核

如何选择正规的期货公司？如何辨别黑平台？
正规期货公司必须满足：
• 经中国证监会批准，持有《期货业务许可证》
• 可在中国期货业协会官网（www.cfachina.org）查询到公示信息
• 目前全国共有约150家持牌期货公司

选择建议：
• 优先选评级A类以上公司（如中信、国泰君安、永安、南华、华泰等）
• 关注交易系统：是否接入CTP主席系统（交易速度与稳定性更好）
• 了解手续费和保证金优惠政策
• 考察客户经理的专业度和服务响应速度

警惕以下黑平台特征：
• 宣称0佣金/0手续费/高额返佣的平台
• 无法在期货业协会官网查询到牌照
• 要求将资金转入个人账户或非银期转账渠道
• 不在国内五大交易所（上期所、大商所、郑商所、中金所、广期所）进行交易

期货手续费怎么算？什么是+1分？
期货手续费由两部分组成：

1. 交易所手续费（固定，全市场统一）
• 按手收取：如螺纹钢约3.6元/手，玉米1.2元/手
• 按合约价值比例：如股指期货约合约价值的万分之0.23

2. 期货公司加收（可协商）
• 默认自助开户：加收交易所的2-5倍
• 通过客户经理协商：可低至交易所+1分（即在交易所基础上只加收0.01元/手，这是行业成本价）

+1分的含义：
指期货公司在交易所手续费基础上仅加收0.01元/手，这是目前行业最低水平。需要在开户前找客户经理书面约定，签署《手续费特别约定书》。

注意事项：
• 部分品种平今仓有单独优惠或加收规则（如股指期货平今仓手续费约为开仓的10倍）
• 监管严禁事后返佣返现，手续费优惠只能在开户签约阶段约定

什么是银期转账？出入金怎么操作？
银期转账是将银行账户与期货保证金账户关联的资金划转系统。

入金（银行→期货账户）：
• 交易软件端操作：登录→银期转账→输入金额和密码→确认
• 银行端操作：手机银行/网银→银期转账→入金
• 入金时间：交易日8:30-15:30（部分银行支持至16:00）
• 入金免费，资金实时到账

出金（期货账户→银行）：
• 交易软件端或银行端操作
• 出金时间：交易日9:00-15:30
• 出金免费

重要提示：
• 当天平仓释放的保证金，需等当日结算完成后（约16:00后）才能出金
• 夜盘时段（21:00-次日2:30）只能入金，不能出金

期货交易时间是怎样安排的？什么是夜盘？
国内期货交易时间分为日盘和夜盘（连续交易）：

日盘交易时间（周一至周五）：
• 上午：9:00-10:15，10:30-11:30（中间休市15分钟）
• 下午：13:30-15:00

夜盘交易时间（品种不同结束时间不同）：
• 21:00-23:00：农产品类（豆粕、玉米、白糖等）
• 21:00-次日01:00：黄金、白银、铜、铝等金属类
• 21:00-次日02:30：原油、天然橡胶、PTA等
• 21:00-23:00（部分品种）：螺纹钢、铁矿石等黑色系

集合竞价时间：
• 有夜盘的品种：夜盘开市前 20:55-21:00（日盘不再集合竞价）
• 无夜盘的品种：日盘开市前 8:55-9:00
• 前4分钟申报，最后1分钟撮合

金融期货（中金所）：
• 股指期货：9:30-11:30，13:00-15:00，无夜盘
• 国债期货：9:30-11:30，13:00-15:15，无夜盘

什么是涨跌停板制度？各品种涨跌停板幅度是多少？
涨跌停板制度是指期货合约在一个交易日中的成交价格不能高于或低于以该合约上一交易日结算价为基准的一定幅度。超过范围的报价视为无效。

涨跌停板幅度因品种而异：
• 商品期货：一般为±4%-±10%（如螺纹钢±4%，原油±10%，黄金±8%）
• 股指期货：正常交易日±10%，最后交易日±20%
• 国债期货：2年期±0.5%，5年期±1.2%，10年期±2%，30年期±3.5%

涨跌停板的作用：
• 防止价格过度波动，给市场冷静期
• 触发涨跌停板后，交易所可能采取调整保证金、扩大涨跌停幅度等措施
• 连续同方向涨跌停板时，交易所会启动强制减仓等风险处置机制

什么是集合竞价？集合竞价的规则是什么？
集合竞价是期货开盘前确定开盘价的过程。

时间安排：
• 前4分钟为申报时间（可以提交和撤销委托）
• 最后1分钟为撮合时间（不可撤单）

撮合原则：
1. 最大成交量原则：选择成交量最大的价格
2. 多个价格满足最大成交量时，取最接近前一交易日结算价的价格
3. 高于成交价的买入申报和低于成交价的卖出申报全部成交
4. 等于成交价的申报按价格优先、时间优先成交

注意事项：
• 集合竞价只能使用限价指令，不能使用市价指令
• 申报价格需在涨跌停板范围内
• 有夜盘的品种，日盘不再进行集合竞价

期货的结算价和收盘价有什么区别？
这是新手容易混淆的概念：

收盘价：当日最后一笔成交价格。

结算价：当日所有成交价格按成交量的加权平均价。这是每日无负债结算的依据。

之所以使用结算价而非收盘价，是为了防止尾盘操纵——如果有人试图在收盘前用少量资金拉高或打压收盘价来影响结算盈亏，加权平均价的机制可以大大增加操纵难度。

此外：
• 股指期货交割结算价 = 最后交易日标的指数最后2小时的算术平均价
• 如果当日无成交，结算价沿用上一交易日的结算价

什么是限价指令和市价指令？
限价指令：指定一个价格，只有在达到该价格或更优价格时才成交。可以控制成本，但不保证一定成交。

市价指令：不指定价格，以当前市场最优价格立即成交。保证成交，但成交价格不可控。
• 市价指令在涨跌停板时可能以涨停板/跌停板价格成交
• 部分品种/交易所有市价指令，部分没有
• 股指期货支持市价指令

止损（盈）指令：当市场价格触及预设触发价时，自动转为市价指令或限价指令执行。是重要的风险控制工具。
19
期货合约到期后怎么办？必须交割实物吗？
个人投资者不能参与实物交割！

对于投机交易的客户：
• 必须在合约最后交易日前自行平仓
• 如果未及时平仓，进入交割月后，交易所有权对不符合交割资质的持仓进行强行平仓
• 自然人客户持仓进入交割月通常会被限制只能平仓不能开仓

交割方式两种：
• 实物交割：商品期货（螺纹钢、铜、大豆等需要现货企业参与的品种），仅限有交割资质的法人客户
• 现金交割：股指期货、国债期货等金融期货，按交割结算价进行现金差额结算

重要提醒：
• 各品种对自然人持仓进入交割月的规定不同，一般交割月前一个月最后一个交易日为自然人最后持仓日
• 建议提前关注交易所公告，在主力合约换月时及时移仓
20
什么是平今仓？平今仓有什么特殊规则？
平今仓是指当日开仓的合约在当日进行平仓操作（即当天开当天平）。

特殊规则：
1. 手续费差异：部分品种平今仓手续费与平昨仓不同
• 股指期货平今仓手续费约为开仓的10倍
• 部分商品期货平今仓有优惠（如黄金、铁矿石平今仓手续费低于开仓）
2. 保证金释放：平今仓释放的保证金当天可用于开仓，但当天不能出金
3. 风控红线：部分交易所对异常高频交易有限制

平昨仓：平掉昨日及之前开仓的持仓，手续费与开仓相同。

期货杠杆是怎么计算的？10%保证金等于几倍杠杆？
期货杠杆倍数 = 1 / 保证金比例

例如：
• 保证金比例10% → 杠杆倍数 = 10倍
• 保证金比例5% → 杠杆倍数 = 20倍
• 保证金比例15% → 杠杆倍数 ≈ 6.7倍

10倍杠杆的实际意义：
如果你投入10万元保证金，可以交易价值100万元的期货合约。价格波动1%，账户盈亏就是1万元（本金的10%），而不是1000元。

杠杆的双面性：
• 价格朝有利方向波动1%：盈利 = 本金的10%
• 价格朝不利方向波动1%：亏损 = 本金的10%
• 反向波动10%：在没有止损的情况下，本金可能全部亏光（爆仓）

建议：新手交易杠杆较高的品种时，实际使用资金不要超过总资金的30%。

什么是每日无负债结算（逐日盯市）？
每日无负债结算制度（逐日盯市/Mark-to-Market）是期货市场核心风控制度。

运作机制：
每个交易日收盘后，交易所按当日结算价对所有持仓进行盈亏计算，并直接划转资金：
• 盈利部分：即时增加到账户权益中
• 亏损部分：即时从账户权益中扣除

当日盈亏计算：
• 平仓盈亏 =（卖出价-买入价）× 合约单位 × 手数
• 持仓盈亏 =（当日结算价-前日结算价）× 合约单位 × 持仓手数

关键影响：
结算后如果账户权益低于维持保证金水平→收到追加保证金通知→需在次日规定时间前补足→否则可能被强行平仓

这确保了期货市场不存在浮亏死扛的空间——亏损每天都会体现在账户中。
23
保证金不够了怎么办？什么情况需要追加保证金？
当账户风险度（持仓占用保证金/账户权益）>100%时，意味着保证金不足，需要立即处理。

你有两种选择：
1. 追加保证金：通过银期转账向期货账户转入资金，使风险度降至100%以下
2. 自行减仓：主动平掉部分持仓，释放保证金

触发追加的情形：
• 持仓方向与市场走势相反，出现较大浮亏
• 交易所临时提高保证金比例（如节假日前、极端行情时）
• 持仓进入交割月，保证金比例逐步提高

时效要求：
• 通常在收到通知后需在次日开盘前补足
• 盘中也可能触发动态风险预警，要求立即追加
• 未及时补足将被强行平仓，损失由投资者自行承担
24
什么是爆仓？爆仓和穿仓有什么区别？
爆仓（强行平仓）：当保证金不足且投资者未在规定时间内补足时，期货公司或交易所将其持仓强行平仓。平仓后账户权益>=0，只是持仓被强制了结。

穿仓：比爆仓更严重的情况。在极端行情下（如连续涨跌停板导致无法及时强平），强行平仓后账户权益变成负数——即投资者不仅亏光了本金，还倒欠期货公司钱。

例如：你投入10万元保证金做多，遇到连续跌停板无法平仓，最终亏损12万元。强平后账户权益为-2万元，这2万元需要你补还给期货公司。

如何避免爆仓/穿仓：
• 坚决不满仓操作，保留30%-50%的备用资金
• 设置并严格执行止损
• 关注交易所保证金调整公告
• 节假日前适当减仓或清仓

25
期货交易有哪些主要风险？
期货交易主要面临以下几类风险：

1. 市场风险：价格波动导致亏损，是最大的风险来源。杠杆放大了价格波动的盈亏幅度。

2. 杠杆风险：保证金交易虽然提高资金效率，但亏损也同比例放大。反向波动10%就可能亏光全部本金。

3. 流动性风险：在极端行情或非主力合约中，可能无法以合理价格及时成交。

4. 强行平仓风险：保证金不足且未及时补足时被强制平仓，错失后续行情反转的机会。

5. 操作风险：下错单、输错数量或方向等人为失误。

6. 系统风险：交易软件、网络故障导致无法及时下单或撤单。

7. 隔夜风险：持有仓位过夜，面临次日开盘跳空的风险。

核心原则：期货交易中，风险管理比盈利更重要。先学会不亏大钱，再追求盈利。
26
什么是止损？如何科学设置止损？
止损是当亏损达到预设价位时自动平仓的风控手段，是期货交易中的保命安全带。

止损设置方法：
• 固定金额止损：每笔交易最大亏损额不超过总资金的1%-2%
• 技术位止损：设置在关键支撑/压力位稍外侧
例：做多螺纹钢，入场价4000元，前期低点在3950元，止损设在3940元
• 波动率止损：以ATR（平均真实波幅）的1.5-2倍作为止损距离
• 时间止损：持仓超过预设时间仍未盈利则平仓

止损纪律：
• 开仓前就必须确定止损位，不可入场后再看情况
• 止损一旦设定，坚决执行，不要移动止损线（除非向盈利方向移动）
• 不要因为感觉马上要反弹了而撤销止损
• 连续止损后暂停交易，冷静复盘再入场
27
什么是强行平仓制度？什么情况下会被强平？
强行平仓（强平）指期货公司或交易所在投资者保证金不足或违规时，未经其同意强制卖出或买入其持仓合约的行为。是期货市场的终极风控机制。

触发强平的常见情形：
1. 保证金不足：账户风险度>100%，且未在规定时间内追加保证金或自行减仓
2. 持仓超限：超过交易所规定的持仓限额
3. 违规交易：异常交易行为、操纵市场等
4. 交割违规：个人投资者进入交割月未平仓

强平执行顺序：
先由会员（期货公司）在开市后第一节自行执行；若未执行完毕，由交易所强制执行。

强平产生的费用和损失由投资者承担。因市场原因无法强平导致的扩大损失也由投资者承担。
28
做期货应该怎样进行资金管理？
资金管理是期货交易中最重要的生存法则：

仓位控制：
• 单笔交易最大亏损不超过总资金的1%-2%
• 总持仓占用保证金不超过总资金的30%-50%
• 单个品种持仓不超过总资金的10%-15%
• 永远不要满仓操作——预留足够资金应对追加保证金

盈亏比管理：
• 每笔交易的预期盈利至少是预期亏损的2-3倍（盈亏比>=2:1）
• 如果止损1000元，目标盈利至少2000元才值得开仓

简单公式：最大开仓手数 =（总资金 × 允许亏损比例）/ 每手止损金额
29
隔夜持仓需要注意什么？
隔夜持仓（持仓过夜）面临的主要风险和注意事项：

主要风险：
• 跳空风险：次日开盘价可能大幅偏离昨日收盘价（因外盘波动、突发事件等），止损单在跳空时可能无法在预设价格成交
• 保证金风险：跳空后保证金可能不足，面临追保压力
• 流动性风险：夜盘成交量少，价格波动可能被放大

建议：
• 新手尽量避免隔夜重仓，收盘前减仓至可承受跳空风险的水平
• 关注夜间可能公布的重要数据和事件（如非农数据、美联储决议、OPEC会议等）
• 节假日前务必减仓：长假期间外盘继续交易，节后可能出现剧烈跳空
• 使用止损止盈条件单进行保护

股指期货有什么特点？适合什么人做？

• IF（沪深300股指期货）：每点300元，覆盖大盘蓝筹
• IH（上证50股指期货）：每点300元，超大盘股

商品期货和金融期货有什么区别？
商品期货和金融期货的核心区别：

| 维度 | 商品期货 | 金融期货 |
|------|---------|---------|
| 标的物 | 实物商品（原油、黄金、大豆等） | 金融工具（指数、国债、汇率等） |
| 交割方式 | 实物交割（个人不能参与） | 现金交割 |
| 定价因素 | 供需关系、天气、库存、政策等 | 宏观经济、货币政策、企业盈利等 |
| 夜盘 | 多数品种有夜盘 | 无夜盘 |
| 开户门槛 | 普通商品期货无额外资金要求 | 50万验资+测试 |
| 参与者 | 产业客户（套保）+投机者 | 机构+专业投资者为主 |

新手建议从商品期货入手，选择流动性好、波动适中的品种（如螺纹钢、甲醇、豆粕），门槛相对较低。

什么是国债期货？有什么特点？
国债期货是以国债为标的的利率期货，在中金所上市，包括：
• TS（2年期国债期货）
• TF（5年期国债期货）
• T（10年期国债期货）
• TL（30年期国债期货）

核心特点：
• 反映市场对未来利率走势的预期——价格上涨=利率下行
• 波动相对较小，风险低于股指期货
• 采用实物交割（一篮子可交割债券）
• 开户门槛与股指期货相同：50万验资+知识测试

主要应用：
• 债券投资者进行利率风险对冲
• 资产配置和久期管理
• 利率方向性交易

什么是期权？期货期权和ETF期权有什么区别？
期权是一种赋予买方在未来特定时间以特定价格买入或卖出标的资产权利（而非义务）的合约。买方支付权利金获得权利，卖方收取权利金承担义务。

看涨期权（Call）：买方有权按约定价格买入标的
看跌期权（Put）：买方有权按约定价格卖出标的

期货期权 vs ETF期权的区别：
• 期货期权的标的是期货合约（如豆粕期权标的是豆粕期货），ETF期权标的是ETF基金
• 期货期权开户门槛低于股指/ETF期权：无需50万资金，通常账户>=1万元即可，但需通过知识测试和模拟交易经历
• 目前国内有40+个商品期权品种和多个股指期权

期权的优势：买方风险有限（最多亏损权利金），收益潜力大；策略灵活多样，可以组合出多种风险收益结构。

什么是套期保值？什么情况下需要套保？
套期保值（Hedging）是指通过在期货市场建立与现货市场相反的头寸，来对冲现货价格波动风险的交易行为。

两类套期保值：
1. 买入套期保值（多头套保）：
适用于担心未来现货价格上涨的企业（如加工商、饲料厂）。在期货市场先买入，锁定采购成本。

2. 卖出套期保值（空头套保）：
适用于持有现货担心价格下跌的企业（如生产商、贸易商、库存方）。在期货市场先卖出，锁定销售价格。

套期保值不是投机：
• 盈亏看期现综合利润，不单独看期货账户盈亏
• 期货端的亏损可以由现货端的盈利对冲，反之亦然
• 关键风险是基差风险——期现价差的不利变化

套期保值适用于有现货背景的实体企业和机构投资者，普通投机者不需要考虑。
36
什么是投机交易？和套保有什么区别？
投机交易是指单纯以获取价差收益为目的的期货交易行为，不持有现货，也不涉及套期保值。

投机 vs 套保的区别：
| 维度 | 投机 | 套期保值 |
|------|------|---------|
| 目的 | 获取价差收益 | 对冲现货价格风险 |
| 现货持仓 | 无 | 有相应的现货头寸 |
| 交易方向 | 单边（纯多或纯空） | 期现反向 |
| 持仓周期 | 短至日内、长至数月 | 与现货周期匹配 |
| 风险特征 | 暴露于全部价格波动 | 仅暴露于基差波动 |
| 盈亏评估 | 只看期货账户 | 看期现综合利润 |

绝大多数个人投资者进行的都是投机交易。投机的关键在于方向判断、风险控制和纪律执行。
37
什么是套利交易？有哪些常见的套利策略？
套利交易是利用不同市场或不同合约之间的价差关系，同时买入和卖出相关合约以获取低风险收益的交易方式。

常见套利类型：

1. 跨期套利（日历价差）：
同一品种不同月份合约之间。例：买入RB2605、卖出RB2610，期待近远月价差回归。

2. 跨品种套利：
产业链相关品种之间。例：买入铁矿石+焦炭、卖出螺纹钢（原料-成品套利）；豆油/豆粕-大豆（压榨套利）。

3. 跨市场套利：
同一品种在不同交易所之间。例：上期所铜 vs LME铜（需考虑汇率、运费等）。

4. 期现套利：
期货与现货之间的基差交易。

套利的特点：风险相对单边投机更低，但收益也相对有限；需要较快的执行速度和较低的手续费。
38
什么是基差？基差对交易有什么影响？
基差 = 现货价格 - 期货价格

基差的正负含义：
• 正基差（现货升水/期货贴水）：现货价格高于期货，通常反映现货供应紧张或市场看跌预期
• 负基差（现货贴水/期货升水）：期货价格高于现货，通常反映持有成本或市场看涨预期

基差对交易者的影响：
• 套期保值者：基差的变动直接影响保值效果
- 卖出套保者希望基差扩大（现货强于期货）
- 买入套保者希望基差缩小（期货强于现货）
• 投机者：基差可以辅助判断市场情绪和多空力量

基差在合约到期时会趋于收敛（归零），这是套利交易的重要理论基础。

新手做期货应该选择什么策略？
给期货新手的建议：

第一步：先学习，再交易
• 至少花1-2个月学习期货基础知识、交易规则、风控方法
• 通过模拟交易熟悉软件操作和下单流程

第二步：选择简单策略
• 新手推荐趋势跟踪策略：顺大势、轻仓位、严止损
• 不要一上来就学复杂的套利或短线高频交易
• 先做流动性好的品种（螺纹钢、甲醇、豆粕等）

第三步：控制交易频率
• 新手最大的敌人是过度交易——T+0的灵活性容易让人停不下来
• 建议每天交易不超过2-3笔，等待高概率机会

第四步：严格纪律
• 每笔交易前写好交易计划：入场理由、止损位、目标位、仓位大小
• 亏损后不要报复性交易——连续亏损2-3次后暂停，冷静复盘

《期货和衍生品法》是什么？对普通交易者有什么影响？
《中华人民共和国期货和衍生品法》于2022年8月1日正式施行，是期货和衍生品市场的基本法，确立了市场运行的基础制度框架。

对普通交易者的核心保护：
• 交易者分类制度：区分普通交易者和专业交易者，对普通交易者给予更强的保护
• 适当性制度：期货经营机构必须了解客户的风险承受能力，推荐合适的产品
• 禁止行为清单：明确禁止欺诈、内幕交易、操纵市场、利益输送等行为
• 民事赔偿机制：完善了交易者维权的法律途径，引入当事人承诺制度
• 多元纠纷解决：建立调解、仲裁等多元纠纷解决机制

此外，该法还明确了持仓限额、当日无负债结算、强行平仓等风控制度的法律地位。
41
期货交易有哪些核心法律制度？
期货市场运行建立在以下核心法律制度之上：

1. 保证金制度：交易者需缴纳合约价值一定比例的资金作为履约担保。

2. 当日无负债结算制度：每日按结算价清算盈亏，资金即时划转。

3. 涨跌停板制度：限制单日价格最大波动幅度。

4. 持仓限额制度：对单一合约的最大持仓量进行限制，防止风险过度集中。如股指期货IF单边持仓限额5000手。

5. 大户报告制度：持仓达到限仓额度80%时需报告资金和头寸情况。

6. 强行平仓制度：保证金不足或违规时强制平仓。

7. 投资者适当性制度：要求期货公司了解客户，将合适的产品卖给合适的投资者。

8. 风险准备金制度：交易所和期货公司提取风险准备金，用于应对市场风险。

9. 信息披露制度：交易所及时公布交易信息，保障市场透明度。

1. 与期货公司协商解决
2. 向中国期货业协会投诉（有调解机制）

4. 申请仲裁（如有仲裁协议）

量化CTA策略通过计算机程序自动执行交易，主要类型：

• 策略失效风险（市场环境变化导致策略不再有效）
• 过拟合风险（回测很好但实盘很差）

期货是不是赌博？和赌博有什么区别？
这是一个常见误解。期货交易与赌博有本质区别：

期货的功能：
• 价格发现：通过公开竞价形成对未来价格的预期，为企业经营提供参考
• 风险管理：企业利用期货套期保值，锁定成本和利润
• 资源配置：引导社会资源向高效领域流动

核心区别：
| 维度 | 期货 | 赌博 |
|------|------|------|
| 经济功能 | 服务于实体经济 | 无经济功能 |
| 风险性质 | 转移/管理已有风险 | 人为创造风险 |
| 法律地位 | 合法合规（受法律保护） | 非法 |
| 可分析性 | 基于基本面/技术面可分析 | 纯概率/运气 |
| 对手方 | 真实的市场参与者 | 庄家/赌场 |

但是：如果不学习、不分析、盲目频繁交易、不设止损——这样做期货和赌博没有区别。关键在于交易者的行为方式，而非工具本身。
48
期货可以满仓操作吗？为什么不能？
强烈不建议满仓操作！

满仓的风险：
1. 没有缓冲空间：期货采用每日无负债结算，价格稍微反向波动就会触发追加保证金
2. 丧失主动权：满仓后任何不利波动都会让你被动——要么追加资金，要么被强制平仓
3. 心理压力巨大：满仓时每个价位波动都牵动神经，容易做出非理性决策
4. 一次失误全盘皆输：期货杠杆下，满仓+方向错误 = 快速爆仓

合理仓位建议：
• 新手：持仓占总资金20%-30%
• 有经验者：不超过50%
• 任何时候保留至少50%资金作为备用保证金

记住：活下来比赚得快更重要。满仓操作的人，最终大概率被市场淘汰。
49
期货亏损了可以像股票一样死扛等反弹吗？
绝对不行！这是股票思维转期货最致命的误区。

股票可以死扛的逻辑：
• 全额交易，没有杠杆，亏损不会放大
• 没有到期日，可以无限期持有
• 股票不会变成负数
• 即使套牢，只要公司不退市，总有回本的可能

期货不能死扛的原因：
• 杠杆放大亏损：10倍杠杆下价格反向10%本金归零
• 每日结算：亏损每天从账户扣钱，扛不住时必须追加保证金
• 会被强平：保证金不足时被强行平仓，根本没机会等反弹
• 合约到期：合约有到期日，必须在到期前平仓
• 可能穿仓：极端行情下不光亏光本金，还会倒欠期货公司

正确做法：严格止损，亏损控制在可承受范围内。期货交易的第一课就是学会认错。
50
期货赚钱快吗？普通人做期货能稳定盈利吗？
实话实说：期货赚钱快，亏钱更快。

现实数据：
• 期货市场长期盈利的散户比例大约在10%-20%左右
• 多数新手在3-6个月内亏损出场
• 杠杆是双刃剑——它创造了暴利故事，也制造了更多爆仓悲剧

能稳定盈利的人有什么特征？
• 把交易当作事业经营，有完整的交易系统
• 严格的风控纪律（止损、仓位管理）
• 持续学习和总结，不把盈亏归结为运气
• 情绪管理能力极强，不被贪婪和恐惧支配
• 通常已经经历了一轮以上的牛熊周期

给普通人的建议：
• 不要抱着暴富心态进入期货市场
• 先把目标设为不亏大钱而非快速翻倍
• 用闲钱交易，不要影响正常生活
• 如果发现持续亏损，停下来反思比继续投入更重要

高收益的期货老师带单群可信吗？
高度警惕！绝大多数都是骗局。

常见骗局套路：
1. 通过社交平台（微信、QQ、抖音等）展示虚假盈利截图吸引关注

辨别方法：
• 凡是承诺稳赚不赔/高收益低风险的——100%是骗局

一句话总结：如果赚钱这么容易，为什么老师不自己做，要带你？

期货技术分析是什么？有哪些常用指标？
技术分析是通过研究历史价格、成交量等市场数据来预测未来价格走势的分析方法。其核心假设是：

1. 市场行为包容一切 — 所有已知信息已经反映在价格中
2. 价格沿趋势运动 — 趋势一旦形成，更可能延续而非反转
3. 历史会重演 — 市场参与者的行为模式具有重复性

常用技术指标分类：
• 趋势指标：移动平均线（MA）、MACD、布林带（Bollinger Bands）— 判断趋势方向和强度
• 摆动指标：RSI（相对强弱指标）、KDJ（随机指标）— 判断超买超卖状态
• 量价指标：成交量、持仓量、OBV（能量潮）— 验证价格走势的可信度
• 形态分析：头肩顶/底、双顶/底、三角形、旗形等 — 判断趋势转折或延续
54
MACD指标怎么看？金叉死叉是什么意思？
MACD（指数平滑异同移动平均线）是最常用的趋势指标之一，由三部分组成：
• DIF线（快线）：12日EMA - 26日EMA
• DEA线（慢线）：DIF的9日EMA
• 柱状线（BAR）：DIF - DEA，红柱表示多头，绿柱表示空头

金叉（买入信号）：DIF线从下方向上穿越DEA线，同时柱状线由绿转红，表示市场由空转多。
死叉（卖出信号）：DIF线从上方向下穿越DEA线，同时柱状线由红转绿，表示市场由多转空。

零轴的意义：
• 零轴上方 = 多头市场，零轴下方 = 空头市场
• 零轴附近金叉 = 较强的买入信号
• 零轴上方死叉 = 可能只是回调，不一定是趋势反转

背离信号：
• 顶背离：价格创新高但MACD未创新高 → 上升动能减弱，可能见顶
• 底背离：价格创新低但MACD未创新低 → 下跌动能减弱，可能见底

注意：MACD在震荡市中容易发出虚假信号，建议结合其他指标和K线形态综合判断。
55
什么是支撑位和阻力位？如何找到它们？
支撑位是价格下跌过程中可能止跌反弹的价格水平，因为该位置有较强的买盘。
阻力位是价格上涨过程中可能遇阻回落的价格水平，因为该位置有较强的卖盘。

寻找支撑/阻力位的方法：
1. 前期高/低点：历史走势中的重要高点和低点，被多次测试的位置更可靠
2. 均线：MA20、MA60、MA120等均线常充当动态支撑/阻力
3. 趋势线：连接两个以上低点（上升趋势线=支撑）或两个以上高点（下降趋势线=阻力）
4. 整数关口：如螺纹钢4000元、原油500元等心理价位
5. 黄金分割：0.382、0.5、0.618等斐波那契回撤位
6. 成交密集区：成交量集中的价格区间，多空博弈充分

实战要点：
• 突破后角色互换：支撑被跌破后变成阻力，阻力被突破后变成支撑
• 多次测试的支撑/阻力位更可靠
• 配合成交量判断：放量突破比缩量突破更可信
56
技术分析和基本面分析有什么区别？应该用哪个？
基本面分析关注影响价格的供需因素：
• 商品期货：产量、库存、进出口、天气、政策、季节性
• 金融期货：宏观经济数据（GDP、CPI、PMI）、货币政策、企业盈利
• 核心问题：价格应该是什么？

技术分析关注价格本身的行为：
• 价格走势、成交量、持仓量、技术指标
• 核心问题：价格正在怎么走？

对比：
| 维度 | 基本面分析 | 技术分析 |
|------|-----------|----------|
| 信息来源 | 经济数据、行业报告 | 价格、成交量 |
| 时间框架 | 中长期 | 短中长皆可 |
| 优势 | 找到价格偏离价值的机会 | 精确把握入场/出场时机 |
| 劣势 | 入场时机难把握，数据滞后 | 信号可能虚假，过于依赖历史 |

建议：两者结合使用
• 用基本面判断大方向和品种选择
• 用技术面确定具体的入场点、止损位和目标位
• 基本面看对的行情，技术面帮你做对
合规交易
4 个问题
57
什么是期货异常交易行为？有哪些类型？
异常交易行为是交易所重点监控的交易行为，一旦触发处理标准，可能被采取监管措施。主要包括以下类型：

1. 自成交：以自己为交易对象，进行自买自卖。目的是制造交易活跃假象或在不同账户间转移资金。

2. 频繁报撤单：频繁申报并撤销申报，不以成交为目的。可能意在试探市场深度或误导其他交易者。

3. 大额报撤单：单笔大额申报后短时间内撤销，可能误导市场判断。

4. 日内开仓量超限：当日开仓量超过交易所规定的限额。

5. 账户组日内开仓超限：实际控制关系账户组合并计算的开仓量超限。

6. 持仓超限：实际控制关系账户合并持仓超过交易所规定的持仓限额。

2026年以来，各交易所对异常交易的查处力度大幅增加，使用AI技术进行实时监控。上期所5月处理异常交易48起，对24个实控账户组限制开仓。
58
什么是实际控制关系账户？需要报备吗？
实际控制关系账户是指由同一人/同一机构实际控制或存在关联关系的多个期货账户。

触发实控关系认定的情形包括：
• 同一自然人/法人实际控制多个账户
• 账户之间存在资金往来或交易一致性
• 账户之间存在亲属关系、合伙关系等
• 账户交易设备（IP/MAC地址）相同

必须报备！根据交易所规定：
• 具有实际控制关系的账户，必须向交易所如实申报
• 未报备或虚假报备的，将被限制开仓、通报批评，可并处50万元以下罚款
• 通过隐瞒实控关系规避持仓限额的，属于严重违规

报备的意义：
• 实控账户组合并计算持仓限额和交易限额
• 防止通过分仓方式规避监管
• 维护市场公平秩序

上期所2026年4月对380组（737个客户）进行了实控关系认定，监管力度不断加强。
59
交易所对违规交易有哪些处罚措施？
交易所对违规行为的处罚分级：

较轻违规（警告/限制类）：
• 电话提示、要求报告情况
• 要求提交书面承诺
• 列入监管关注名单
• 约见谈话
• 限制开仓交易1个月以内

较重违规（纪律处分类）：
• 通报批评、公开谴责
• 暂停开仓交易1-12个月
• 取消会员资格
• 宣布为"市场禁止进入者"
• 罚款：5万-100万元，或违规所得1-5倍

特别严重的后果：
• 操纵市场、内幕交易：移送证监会立案调查
• 相互交易转移资金（对敲）：非法获利超40万元的案例曾被暂停开仓12个月
• 进入交割月的违规持仓被强平：盈利部分将被没收

合规建议：日常交易中保持警惕，不触碰监管红线。如收到交易所询问或调查，应积极配合，如实回复。

普通投资者如何避免异常交易？（避坑指南）
普通投资者避免异常交易的实操建议：

1. 不要多账户操作
• 不要用家人/朋友账户同时交易同一品种
• 如确实有多个账户，主动向期货公司申报实控关系

2. 不要频繁撤单
• 下单前确认价格和数量，避免反复修改撤单
• 程序化交易者注意控制撤单率

3. 注意持仓限额
• 了解各品种的持仓限额（如股指IF单边限仓5000手）
• 接近限额时主动减仓

4. 注意日内开仓限额
• 部分品种对日内开仓量有硬性上限
• 程序化策略需内置开仓量监控

5. 进入交割月及时平仓
• 自然人客户需在交割月前平仓
• 关注交易所公告中的交割月持仓规定

6. 不要代客理财
• 没有资质的个人代他人操作期货账户属于违规
• 不参与任何形式的"代操盘""喊单""跟单"

一句话：合规交易就是保护自己，不要试图钻规则的空子。

什么是量化CTA策略？和手工交易有什么不同？
量化CTA策略（Commodity Trading Advisor）是利用计算机程序和数学模型，在期货市场自动执行交易的策略。CTA名称源于美国商品期货交易顾问的注册类别。

主要策略类型：
• 趋势跟踪：识别并跟随市场趋势，是最主流策略
• 统计套利：利用品种间的统计关系进行对冲交易
• 均值回归：押注价格回归历史均值
• 高频交易：利用极短时间内的价差/订单流获利
• 多因子策略：基于多个量化因子（动量、基差、库存等）构建组合

量化 vs 手工交易：
| 维度 | 量化CTA | 手工交易 |
|------|---------|----------|
| 决策方式 | 程序化、规则化 | 主观判断 |
| 情绪干扰 | 无 | 容易受贪婪/恐惧影响 |
| 同时监控品种 | 数十个 | 1-3个 |
| 执行力 | 严格 | 可能犹豫/冲动 |
| 策略容量 | 受限于模型和资金 | 受限于个人精力 |
| 黑箱风险 | 策略逻辑可能不透明 | 不存在 |

CTA策略在2026年表现亮眼，部分基金年内收益超12%。但量化交易也有策略失效、过拟合等风险，并非稳赚不赔。

什么是策略回测？回测有哪些常见的陷阱？
策略回测（Backtesting）是将交易策略在历史数据上模拟运行，评估其表现的过程。这是量化策略开发中的核心验证环节。

回测的核心指标：
• 年化收益率、最大回撤、夏普比率
• 胜率（盈利交易占比）、盈亏比
• 最大连续亏损次数

常见陷阱（新人必看）：

1. 过拟合（最致命的陷阱）
策略在历史数据上表现极好，但实盘效果很差。原因是参数过度优化到拟合了历史噪音。表现为回测曲线完美向上、几乎无回撤。

2. 未来函数/前视偏差
回测时不小心使用了当时还不知道的数据。如用当日收盘价作为当日开盘交易的信号。

3. 幸存者偏差
只用当前还在交易的品种做回测，忽略了已退市或失败的品种。

4. 忽略交易成本
回测时未计入手续费、滑点、冲击成本，导致收益虚高。

5. 过少的样本
仅回测几个月的数据不足以验证策略有效性。

建议：回测至少覆盖3-5年数据（包含牛熊周期），做样本外检验，保持策略逻辑简洁，多个维度交叉验证。

什么是移仓换月？怎样正确操作？
移仓换月（展期/Rollover）是指将即将到期的期货合约平仓，同时在更远月份的合约上开仓，以此延续持仓的操作。

为什么要移仓换月？
• 期货合约有到期日，到期前必须平仓或交割
• 个人投资者不能参与实物交割
• 进入交割月后保证金提高、流动性下降
• 主力合约会切换到下一个活跃月份

操作步骤：
1. 关注交易所公告，了解主力合约切换时间（通常在交割月前一个月）
2. 观察新旧主力合约的成交量和持仓量变化
3. 先平旧仓：将即将到期合约的持仓平掉
4. 后开新仓：在远期活跃合约上按原方向重新建仓
5. 注意新旧合约的价差（升贴水），这会影响实际交易成本

移仓换月的成本：
• 新合约价格可能高于旧合约（升水结构）→ 多头移仓有额外成本
• 新合约价格可能低于旧合约（贴水结构）→ 空头移仓有额外成本
• 还需要支付两次交易手续费

建议在主力合约切换前3-5个交易日完成移仓，避免在流动性下降时被动操作。

什么是基差交易？怎么利用基差获利？
基差交易（Basis Trading）是利用现货价格与期货价格之间的价差变化来获利的交易策略。

基差 = 现货价格 - 期货价格
• 正基差（现货升水）：现货 > 期货
• 负基差（现货贴水）：现货 < 期货

基差交易的核心逻辑：
在期货合约临近到期时，基差会收敛趋近于零（期现价格趋于一致）。基差交易就是押注这个收敛过程。

典型策略：
1. 正向套利：期货价格远高于现货（深度升水）→ 买入现货 + 卖出期货 → 等待收敛获利
2. 反向套利：现货价格远高于期货（深度贴水）→ 卖出现货 + 买入期货 → 等待收敛获利

基差交易的特点：
• 风险相对单边投机更小
• 需要同时参与现货和期货市场
• 涉及仓储、物流、资金成本等实际操作
• 更多是机构和企业参与，个人投资者门槛较高

个人投资者关注基差的意义：
• 基差变化可以辅助判断市场多空情绪
• 深度升贴水可能预示行情转折
• 套保客户需要密切关注基差对保值效果的影响

期货市场的参与主体有哪些？各自扮演什么角色？
期货市场的参与主体分为以下几类：

1. 套期保值者（Hedgers）
• 身份：生产企业、贸易商、加工企业、农户等
• 目的：对冲现货价格波动风险，锁定成本或利润
• 特点：期现结合操作，不追求期货端盈利
• 举例：钢厂卖出螺纹钢期货锁定销售价格，油脂厂买入大豆期货锁定原料成本

2. 投机者（Speculators）
• 身份：个人投资者、私募基金、自营交易员
• 目的：通过预测价格走势赚取价差
• 特点：不持有现货，单边做多或做空
• 作用：为市场提供流动性，承担套保者转移的风险

3. 套利者（Arbitrageurs）
• 身份：专业交易团队、量化基金
• 目的：利用价差关系获取低风险收益
• 特点：跨期/跨品种/跨市场同时买卖
• 作用：促进合理价差形成，提高市场效率

4. 做市商（Market Makers）
• 身份：期货公司风险管理子公司、专业做市机构
• 目的：通过双边报价赚取买卖价差
• 作用：提供持续流动性，缩小买卖价差

5. 交易所与清算机构
• 提供交易平台和中央对手方清算服务
• 负责一线监管和风险控制

什么是含权贸易？期货期权如何服务实体经济？
含权贸易是将期权嵌入现货贸易合同中的一种创新业务模式。由期货公司风险管理子公司为企业量身定制。

举个例子：
一家饲料厂担心未来豆粕涨价，但又不想支付全额保证金去做期货套保。期货风险管理子公司可以为它设计一份含权采购合同：
• 约定未来3个月采购豆粕，保底价3000元/吨
• 如果市场价涨到3500元，饲料厂仍按3000元采购（期权保护）
• 如果市场价跌到2800元，饲料厂按2800元采购（享受降价）
• 饲料厂支付一笔权利金（比直接做期货套保的资金占用小很多）

含权贸易的价值：
• 企业不需要懂期货期权操作，只需正常的采购/销售合同
• 风险管理子公司承担专业对冲角色
• 降低中小企业参与风险管理的门槛
• 实现"保底价+享受有利行情"的灵活方案

这是期货市场服务实体经济的重要体现之一，也是近年来期货行业大力发展的业务方向。目前"保险+期货"、"含权贸易"等模式已被广泛应用于农产品、有色金属、能源化工等产业领域。
